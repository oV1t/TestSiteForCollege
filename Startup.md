# Послідовність запуску проєкту

Цей документ містить інструкції щодо розгортання та запуску локальної копії проєкту.

## 1. Налаштування Бекенду (Python / FastAPI)

Перейдіть до директорії `backend`:
```bash
cd backend
```

### Крок 1: Встановлення залежностей
Рекомендується використовувати віртуальне середовище. Встановіть необхідні бібліотеки:
```bash
pip install -r requirements.txt
```

### Крок 2: Ініціалізація бази даних
Запустіть скрипт для створення таблиць та наповнення бази даними (студенти, дисципліни, адмін):
```bash
python seed.py
```
*Це створить файл `elective_disciplines.db` у папці `backend`.*

### Крок 3: Запуск сервера
Запустіть сервер FastAPI за допомогою uvicorn:
```bash
uvicorn main:app --reload
```
API буде доступне за адресою: [http://127.0.0.1:8000](http://127.0.0.1:8000)

---

## 2. Налаштування Фронтенду (Vue / Vite)

Відкрийте **нове вікно терміналу** та перейдіть до директорії `frontend`:
```bash
cd frontend
```

### Крок 1: Встановлення залежностей Node.js
Встановіть необхідні пакети:
```bash
npm install
```

### Крок 2: Запуск у режимі розробки
Запустіть Vite dev-сервер:
```bash
npm run dev
```
Фронтенд буде доступний за адресою: [http://localhost:5173](http://localhost:5173) (або іншою, яку вкаже термінал).

---

## Важливі зауваження
- Переконайтеся, що у вас встановлені **Python 3.10+** та **Node.js 18+**.
- Бекенд та Фронтенд повинні працювати одночасно у різних вікнах терміналу.
